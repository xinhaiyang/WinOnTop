# Window On Top

一个轻量级的 Windows 桌面应用程序，用于将任意窗口置顶或取消置顶。

## 功能特性

- **拖拽置顶**：拖动手型图标到目标窗口，释放鼠标即可置顶
- **快捷键操作**：支持多个全局热键快速操作窗口
  - `Ctrl+F8`：切换当前活动窗口的置顶状态
  - `Ctrl+F9`：将当前活动窗口设置为指定透明度
  - `Ctrl+F10`：切换幽灵模式（半透明 + 点击穿透）
- **窗口透明度**：通过滑块调节透明度（20%-100%），实时应用到目标窗口
- **幽灵模式**：窗口半透明（50%）且点击穿透，可看到下层窗口内容
- **系统托盘**：最小化到系统托盘，不占用任务栏空间
- **全局热键**：即使程序窗口不在前台，快捷键依然有效

## 系统要求

- Windows 10 或 Windows 11
- .NET 10.0 或更高版本

## 使用方法

### 方式一：拖拽置顶

1. 按住界面中的**手型图标**
2. 拖动到想要置顶的窗口上
3. 释放鼠标，该窗口将被置顶

### 方式二：快捷键操作

#### 置顶/取消置顶

1. 点击想要操作的窗口，使其成为活动窗口
2. 按下 `Ctrl+F8`
3. 窗口将在"置顶"和"取消置顶"之间切换

#### 设置窗口透明度

1. 在主界面调节透明度滑块（20%-100%）
2. 点击想要操作的窗口，使其成为活动窗口
3. 按下 `Ctrl+F9`，窗口将应用设置的透明度

#### 幽灵模式（半透明 + 点击穿透）

1. 点击想要操作的窗口，使其成为活动窗口
2. 按下 `Ctrl+F10`
3. 窗口将进入幽灵模式（50%透明度 + 点击穿透）
4. 再次按下 `Ctrl+F10` 可恢复正常状态

### 系统托盘操作

- **双击托盘图标**：显示主窗口
- **右键托盘图标**：打开菜单，包含以下选项：
  - **显示**：显示主窗口
  - **透明度**：子菜单可选择 100%、80%、60%、40% 透明度
  - **幽灵模式 (Ctrl+F10)**：快速切换幽灵模式
  - **退出**：退出程序
- **点击最小化按钮**：窗口隐藏到托盘

## 编译运行

### 开发环境运行

```bash
dotnet run
```

### 发布可执行文件

```bash
dotnet publish -c Release -r win-x64 --self-contained true
```

发布后的文件位于 `bin\Release\net10.0-windows\win-x64\publish\` 目录下。

## 技术实现

- 使用 Windows API (`SetWindowPos`) 实现窗口置顶
- 使用 `RegisterHotKey` 注册全局热键
- 使用 `NotifyIcon` 实现系统托盘功能
- 使用 `WindowFromPoint` 获取鼠标位置下的窗口句柄
- 使用 `SetLayeredWindowAttributes` 实现窗口透明度
- 使用 `WS_EX_TRANSPARENT` 样式实现点击穿透效果（幽灵模式）

## 注意事项

- 某些系统窗口或受保护的窗口可能无法置顶
- 程序需要以普通用户权限运行即可，无需管理员权限
- 全局热键 `Ctrl+F8` 在程序运行期间始终有效
